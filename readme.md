## Requirements

### C++

For running C++ you need to install g++ in an unix environment

### C#

For running C# you need to install the latest .NET Core SDK (2.2 or higher)

## Running the Program

### C++

```
cd cpp
g++ main.cpp -std=c++11 -lpthread
./a.out
```

### C#

```
cd cs
dotnet restore
dotnet run
```
